# Who Uses Commerce Gateway

---

## Production

### [Better Data](https://betterdata.co)

AI-native Commerce Chain Optimization platform serving Healthcare, Pharma, Retail, Manufacturing, Food & Beverage, and Construction.

Commerce Gateway powers the AI agent interface layer in the Better Data platform — connecting Claude, ChatGPT, and Grok to live commerce chain data and operations via Remote MCP/SSE transport. All tool calls from Better Data's AI advisors route through the open gateway runtime.

Better Data created and maintains Commerce Gateway.

## Add Your Project

Building something with Commerce Gateway? Open a pull request to add it here.

We list:
- **Production deployments** — Commerce Gateway running in a live system
- **Significant OSS projects** — Projects that extend or integrate the gateway
- **Compatible connectors** — Third-party connectors for platforms not in the core package
